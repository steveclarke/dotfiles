# useCurrentElement

Get the DOM element of current component as a ref.

**Package:** `@vueuse/core`
**Category:** Component

## Usage

```ts
import { useCurrentElement } from '@vueuse/core'

const el = useCurrentElement() // ComputedRef<Element>
```
