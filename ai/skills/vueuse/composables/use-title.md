# useTitle

Reactive document title.

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { useTitle } from '@vueuse/core'

const title = useTitle()
console.log(title.value) // print current title
title.value = 'Hello' // change current title
```
