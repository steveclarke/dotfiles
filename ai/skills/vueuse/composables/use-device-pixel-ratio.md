# useDevicePixelRatio

Reactively track

**Package:** `@vueuse/core`
**Category:** Sensors

## Usage

```ts
import { useDevicePixelRatio } from '@vueuse/core'

const { pixelRatio } = useDevicePixelRatio()
```

## Returns

| Name       | Type         |
| ---------- | ------------ |
