# useFps

Reactive FPS (frames per second).

**Package:** `@vueuse/core`
**Category:** Sensors

## Usage

```ts
import { useFps } from '@vueuse/core'

const fps = useFps()
```

## Options

| Option | Type     | Default | Description                          |
| ------ | -------- | ------- | ------------------------------------ |
| every  | `number` | 10      | Calculate the FPS on every x frames. |
