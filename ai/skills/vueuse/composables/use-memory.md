# useMemory

Reactive Memory Info.

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { useMemory } from '@vueuse/core'

const { isSupported, memory } = useMemory()
```

## Returns

| Name        | Type                        |
| ----------- | --------------------------- |
| isSupported | `useSupported`              |
| memory      | `deepRef&lt;MemoryInfo&gt;` |
