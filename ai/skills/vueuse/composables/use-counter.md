# useCounter

Basic counter with utility functions.

**Package:** `@vueuse/shared`
**Category:** Utilities

## Usage

```ts
import { useCounter } from '@vueuse/core'

const { count, inc, dec, set, reset } = useCounter()
```

## Returns

| Name  | Type         |
| ----- | ------------ |
| count | `shallowRef` |
| inc   | `Ref`        |
| dec   | `Ref`        |
| get   | `Ref`        |
| set   | `Ref`        |
| reset | `Ref`        |
