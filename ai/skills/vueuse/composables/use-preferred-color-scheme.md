# usePreferredColorScheme

Reactive prefers-color-scheme media query.

**Package:** `@vueuse/core`
**Category:** Browser

## Usage

```ts
import { usePreferredColorScheme } from '@vueuse/core'

const preferredColor = usePreferredColorScheme()
```
